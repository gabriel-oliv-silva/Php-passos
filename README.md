
#  Projetos de Aprendizado em PHP & Symfony

Este repositório reúne projetos desenvolvidos com **PHP** e **Symfony**, com foco exclusivamente em **aprendizado, prática e evolução técnica**.

----------

##  Objetivo

O principal objetivo deste repositório é:

-   Praticar conceitos de desenvolvimento backend
    
-   Explorar o ecossistema do Symfony
    
-   Aplicar boas práticas de arquitetura
    
-   Evoluir continuamente como desenvolvedor
    

----------

##  Tecnologias Utilizadas

-   PHP
    
-   Symfony 8

-   Twig

## 📂 Estrutura

Cada projeto está organizado em sua própria pasta, contendo:

-   Código-fonte

----------

##  Como rodar os projetos

Clone o repositório:

```bash
git clone https://github.com/gabriel-oliv-silva/nome-repositorio.git

```

Acesse a pasta do projeto desejado:

```bash
cd nome-do-projeto

```

Instale as dependências:

```bash
composer install

```

Configure o ambiente:

-   Ajuste o arquivo `.env` (principalmente o `DATABASE_URL`)
    

Execute o projeto:

```bash
symfony server:start

```

ou:

```bash
php -S localhost:8000 -t public

```
